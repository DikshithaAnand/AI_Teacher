
# AI Assistant (Full Structure)

This project demonstrates a professional AI assistant using:
- Streamlit (UI)
- Backend API layer
- Free instruction-tuned LLM (FLAN-T5)

## Structure
frontend/  -> UI logic (Streamlit)
backend/   -> LLM API handling
app.py     -> Entry point

## Requirement
Set environment variable:
HF_API_TOKEN=your_huggingface_token
